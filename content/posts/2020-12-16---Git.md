### Git 
- 컴퓨터 파일의 변경사항을 추적하고 유저들간에 해당 파일에 대한 작업을 조율하기 위한 분산 버전 관리 시스템
- 소스 코드 관리에 주로 사용한다.
- 분산 버전 관리 시스템으로서 빠른 수행 속도에 중점을 두고 있는 것이 특징
- 데이터 무결성, 분산, 비선형 워크플로를 지원한다.
